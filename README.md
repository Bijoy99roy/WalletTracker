# Solana Wallet Tracker

A modern wallet tracker for solana ecosystem

![alt text](image.png)

## Backend

- Helius API
- BirdEye API
- Express.js
- @solana/web3.js

## Frontend

- React
- ShadCn
- MagicUi

```cmd
To run add a .env file add proper credentials for:
Helius
PostgresDB
BirdEye

Set Env to DEV

Then you can run both frontend and backend individually
```
